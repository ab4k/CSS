Q. Write a detailed article on css selectors, pseudo selectors and their uses with example.

A. CSS Selectors are addresses of the targets which we have to style using css properties.

Universal Selector

There are various ways to select the html elements, so we can style the elements as per our requirement. We can select the html elements using individual selctors like the tag names, h1, p, div, span, etc. In this case all the elements having the same tag will get selected and the style will be applied. If we want to select the entire html we can use universal selector, to select the entire page at a time we have to use * (asterisk).
